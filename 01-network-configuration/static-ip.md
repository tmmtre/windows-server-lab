# Configure Static IP - Windows Server

## Objective
Configure a static IP address for the Windows Server machine to ensure
stable network communication inside the lab environment.

## Steps

1. Open Settings
2. Go to Network & internet
3. Then Advance network settings
4. Select Ethernet in Network adapters
5. Click Edit on More adapter options
6. Double click on Internet Protocol Version 4 (IPv4)

## Configuration used in the lab

IP Address: 10.0.2.15  
Subnet Mask: 255.255.255.0  
Default Gateway: 10.0.2.2  
DNS Server: 10.0.2.15 (still not installed)

## Result

The server now uses a static IP address required for domain services.
