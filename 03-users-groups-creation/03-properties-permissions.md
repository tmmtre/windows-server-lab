# Properties and permissions - Windows Server

## Objective
To manage user account properties and configure folder permissions for users or groups.

## Steps

### Properties
1. Open Active Directory Users and Computers
2. Double-click the user you want to edit
3. Go to the Account tab, change previously set options or set logon hours if needed, e.g., for backup accounts

### Permissions
1. Navigate to a folder on a computer or server, e.g., Documents
2. Right-click the folder, select Properties, then go to the Security tab
3. Click Edit, then search for the user or group you want to add
4. After adding, choose the permissions and click OK
