# Create users - Windows Server

## Objective
To create user accounts in Active Directory for management and authentication purposes.

## Steps

1. Open Server Manager, go to Tools, and select Active Directory Users and Computers
2. In the left pane, navigate to your domain, right-click it, and select New → Organizational Unit
3. Give the OU a descriptive name for the group of users you want to manage and click OK
4. Inside the newly created OU, right-click and select New → User
5. Fill in the user’s First Name, Last Name, and User logon name (must be unique), then click Next
6. Set a password that meets your domain’s password policy. Optionally, check whats better for that User
7. Click Next, review the settings, and click Finish to create the user
