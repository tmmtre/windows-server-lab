# Create groups - Windows Server

## Objective
To create groups in Active Directory for better organization and to manage permissions efficiently.

## Steps

1. Open Active Directory Users and Computers
2. Right-click your domain and select New → Organizational Unit. Name it, for example, Groups and click OK
3. Inside the newly created OU, right-click and select New → Group
4. Give the group a name, choose a group scope, and select a group type. Click OK
5. Select the group you created, right-click it, and choose Properties
6. Go to the Members tab and click Add
7. Search for the user(s) you want to add, select them, and click OK

## Configuration used in the lab
Organizational Unit: Groups
Group name: IT Admins
