# Install AD Domain Services and DNS Server - Windows Server

## Objective
Install AD Domain Services and DNS Server to start the home lab.

## Steps

1. In the Server Manager go to Manage
2. Click Add Roles and Feature
3. Click Next, select Role-based or Feature-based installation and click Next
4. Select the server and Next again
5. Check the roles Active Directory Domain Services and DNS Server
6. Next until Confirmation and then Install

## Result

The two Roles are now installed.
