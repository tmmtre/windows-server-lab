# Install DHCP Server - Windows Server

## Objective
Install DHCP Server.

## Steps

1. In the Server Manager go to Manage
2. Click Add Roles and Feature
3. Click Next, select Role-based or Feature-based installation and click Next
4. Select the server and Next again
5. Check the role DHCP Server
6. Next until Confirmation and then Install
7. Go to the flag icon and click Compleate DHCP configuration
8. Click Next, choose a User Name then Commit and Close

## Result

The Roles is now installed.
